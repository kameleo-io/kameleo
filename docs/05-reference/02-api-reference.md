All endpoints below are available through the official Kameleo SDKs. Prefer using an SDK for typed models, auth handling, and forward compatibility instead of crafting raw HTTP requests manually.

OpenAPI v3 specification: [swagger.json](/static/swagger.json)

Official SDK packages:

- Python: [kameleo-local-api-client (PyPI)](https://pypi.org/project/kameleo-local-api-client/)
- JavaScript / TypeScript: [@kameleo/local-api-client (npm)](https://www.npmjs.com/package/@kameleo/local-api-client)
- C#: [Kameleo.LocalApiClient (NuGet)](https://www.nuget.org/packages/Kameleo.LocalApiClient)
